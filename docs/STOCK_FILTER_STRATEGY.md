## 一、现状问题

| 项目 | 现状 |
|------|------|
| 主板股票总数 | 约 4000+ 只 |
| ST股票 | 约 200-300 只 |
| 当前刷新数量 | 200 只 |
| 筛选策略 | 仅市值排序，无分层 |

**核心问题**：
- 只分析市值前200只，容易遗漏中小市值优质标的
- 无硬性排除层，ST股等垃圾数据干扰分析
- 无动态调整机制，股票池僵化

## 二、筛选维度体系

### 2.1 硬性排除层（零容忍，直接剔除）

以下股票**永久不进入分析池**：

| 类别 | 标准 | 原因 |
|------|------|------|
| ST/*ST/SST | 状态包含 ST/*ST/SST | 基本面恶化，规则特殊 |
| 退市整理期 | 处于退市整理期 | 无分析价值 |
| 次新股 | 上市不足 **60 个交易日** | 波动异常，历史数据不足 |
| 僵尸股 | 连续 **20 日**成交额 < **500 万** | 数据失真，无法执行策略 |
| 长期停牌 | 停牌超过 **30 个交易日** | 数据断层 |
| 价格过低 | 股价 < **3 元** | 低价垃圾股嫌疑 |

### 2.2 质量准入层（主板核心门槛）

通过硬性排除后，满足以下条件才可进入分析池：

| 维度 | 标准 | 说明 |
|------|------|------|
| **市值门槛** | 流通市值 ≥ **5 亿** | 规避"壳"价值干扰 |
| **流动性门槛** | 近 20 日日均成交额 ≥ **2000 万** | 确保技术分析有效性 |
| **财务健康** | 满足以下任意 **2 条**：净利润为正、营收同比增长 ≥ 0、资产负债率 ＜ 70% | 避免暴雷 |
| **价格门槛** | 股价 ≥ **3 元** | 排除低价垃圾股 |

### 2.3 分层维度

| 维度 | 权重 | 说明 |
|------|------|------|
| **市值** | 高 | 流动性好，代表性强 |
| **指数成分** | 高 | 沪深300/中证500/中证1000成分股 |
| **换手率** | 中 | 市场关注度高 |
| **近期动量** | 中 | 涨幅/成交量异常可能是机会 |
| **机构覆盖** | 中 | 研报覆盖、北向资金持股 |

## 三、分层策略（四层架构）

```
┌─────────────────────────────────────────────────────────────┐
│  第一层：核心池（约 300 只）                                  │
│  - 沪深300成分 + 中证500成分 + 日均成交额前300                │
│  - 高频更新：每日 + 盘中快照                                   │
│  - 全维度分析：基本面 + 技术面 + 舆情                           │
├─────────────────────────────────────────────────────────────┤
│  第二层：活跃池（约 500 只）                                   │
│  - 中证1000成分 + 近20日换手率Top20% + 行业龙头               │
│  - 中频更新：每日收盘后                                       │
│  - 标准分析：基本面 + 简化技术指标                             │
├─────────────────────────────────────────────────────────────┤
│  第三层：观察池（约 1000 只）                                  │
│  - 满足准入条件的主板股票（排除前两层）                        │
│  - 低频更新：每周 2-3 次                                      │
│  - 轻量扫描：基础监控 + 异动触发                              │
├─────────────────────────────────────────────────────────────┤
│  第四层：临时池（事件驱动）                                    │
│  - 涨停/跌停、巨量异动、公告、舆情触发                        │
│  - 按需更新：一事一议                                          │
│  - 临时深度分析，完成后回归原池                                │
└─────────────────────────────────────────────────────────────┘
```

**覆盖总量：约 1800 只，占主板总量 40-45%**

## 四、核心筛选规则

### 4.1 核心池入选标准（满足任一即可）

```
1. 沪深300成分股
2. 中证500成分股
3. 近 20 日日均成交额排名前 300
4. 日均成交额 ≥ 1 亿 + 财务稳健（ROE > 5%）
```

### 4.2 活跃池入选标准（满足任一即可）

```
1. 中证1000成分股
2. 近 20 日换手率排名 Top 20%
3. 每个一级行业市值最大的 2-3 只（即使不在前500）
4. 近 30 日涨幅排名前 50 的非核心池股票
```

### 4.3 观察池入选标准

```
1. 满足质量准入条件
2. 未被核心池/活跃池覆盖
3. 日均成交额 ≥ 1000 万
```

### 4.4 动态晋级条件

| 从 | 到 | 触发条件 |
|----|----|---------|
| 观察池 | 活跃池 | 近30日涨幅持续超过活跃池平均水平 |
| 观察池 | 核心池 | 纳入沪深300/中证500 |
| 活跃池 | 核心池 | 日均成交额进入前 300 |
| 任意池 | 临时池 | 涨停、巨量异动（换手率>20%）、重大公告 |

### 4.5 降级/剔除条件

| 池 | 条件 | 动作 |
|----|------|------|
| 核心/活跃池 | 连续 15 日成交额低于门槛 50% | 降级至下一层 |
| 核心/活跃池 | 被ST/*ST | 立即移出，进入监控 |
| 任意池 | 触发硬性排除条件 | 移出分析池 |
| 观察池 | 连续 30 日成交额 < 500 万 | 移出，仅保留基础行情 |

## 五、更新频率策略

| 池 | 更新频率 | 数据粒度 | 算力配额 |
|----|---------|---------|---------|
| 核心池 | **每日 + 盘中快照** | 分钟级/日线 | 50% |
| 活跃池 | **每日收盘后** | 日线 | 30% |
| 观察池 | **每周 2-3 次** | 日线 | 15% |
| 临时池 | **事件触发** | 按需 | 5% |

**算力配额**：核心池占用一半算力，确保核心标的分析质量

## 六、指数成分联动

### 6.1 自动同步机制

- **沪深300、中证500、中证1000** 成分股每半年调整（6月/12月）
- 系统在调整生效日前一天自动更新核心池/活跃池
- 数据源：中证指数官网 或 Tushare 指数成分 API

### 6.2 成分股权重利用

- 沪深300成分股默认进入核心池
- 中证500成分股默认进入活跃池
- 中证1000成分股优先进入活跃池

## 七、事件驱动临时池

### 7.1 触发条件

| 事件类型 | 触发条件 | 分析深度 |
|---------|---------|---------|
| 涨停异动 | 单日涨幅 > 9.5% 或 跌幅 > 7% | 深度分析 |
| 巨量异动 | 换手率 > 20% 或 成交额暴增 3 倍 | 深度分析 |
| 公告驱动 | 业绩预增、重组、股权变更等 | 按需分析 |
| 舆情驱动 | 龙虎榜、财经新闻出现 | 快速扫描 |

### 7.2 处理流程

```
触发事件 → 临时加入临时池 → 深度分析 → 生成报告
    ↓
结论：持有/不持有 → 次日回归原池
```

## 八、配置化设计

```python
STOCK_FILTER_CONFIG = {
    # 硬性排除
    "exclude": {
        "st_stock": True,                    # 排除ST股
        "new_stock_days": 60,               # 次新股排除：上市60个交易日
        "min_daily_volume": 500,            # 万元，连续20日低于此值排除
        "suspended_days": 30,                # 停牌超过此天数排除
        "min_price": 3.0,                    # 元，股价低于此值排除
    },

    # 质量准入
    "admission": {
        "min_market_cap": 5,                # 亿元，流通市值门槛
        "min_daily_volume_20d": 2000,       # 万元，近20日日均成交额
        "financial_health": {                # 财务健康（满足2/3）
            "positive_profit": True,
            "revenue_growth_gte_0": True,
            "debt_ratio_lt_70": True,
        },
    },

    # 分层阈值
    "tiers": {
        "core": {
            "size": 300,                    # 目标数量
            "index_components": ["hs300", "zz500"],  # 指数成分
            "min_volume_rank": 300,          # 日均成交额排名
            "update_frequency": "daily_plus",  # 每日+盘中
        },
        "active": {
            "size": 500,
            "index_components": ["zz1000"],
            "min_turnover_rank": 20,         # 换手率Top20%
            "update_frequency": "daily",
        },
        "observe": {
            "size": 1000,
            "min_volume": 1000,             # 万元，日均成交额门槛
            "update_frequency": "weekly_2",  # 每周2次
        },
    },

    # 动态调整
    "rebalance": {
        "downgrade_volume_threshold": 0.5,   # 成交额低于门槛50%持续N日降级
        "downgrade_days": 15,
        "upgrade_momentum_streak": 5,        # 连续N日动量超标晋级
    },

    # 事件驱动
    "event_driven": {
        "limit_up_pct": 9.5,                # 涨停触发
        "limit_down_pct": -7.0,             # 跌停触发
        "turnover_spike": 20,               # 换手率暴增%
        "volume_spike": 3,                  # 成交额暴增倍数
    }
}
```
