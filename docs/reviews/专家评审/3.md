# 策略优化设计方案评审报告

## 一、方案概述

该方案旨在建立系统的策略评测和优化流程，验证稳健型策略在 2024-2025 年数据上的表现。方案结构完整，包含现状分析、优化目标、实施流程、技术实现和验证方案等核心部分。

---

## 二、方案优势

### 2.1 目标明确，定位清晰

方案明确聚焦于「稳健型策略」，持仓 5-8 只、单只仓位 8-15%、止损 -10%、最大回撤容忍 15%，这些参数设定符合稳健型策略的核心特征。同时将「胜率」和「盈亏比」作为核心指标，与策略定位高度一致。

### 2.2 流程设计科学合理

三阶段优化流程逻辑清晰、递进合理。**阶段一**通过策略横向对比筛选最优策略类型；**阶段二**对最优策略进行参数网格搜索；**阶段三**深入分析因子贡献。这种「筛选—优化—归因」的三层递进结构能够有效降低优化成本，避免在表现一般的策略上浪费调参资源。

### 2.3 验证方式严谨规范

采用训练集/测试集分离的设计（2024 年训练、2025 年测试），并设定明确的筛选条件（最大回撤 ≤ 15%、年化收益 > 10%、交易次数 ≥ 50），有效避免过拟合风险，提升结果可信度。

### 2.4 输出格式规范

方案定义了清晰的 JSON 输出格式，包含策略对比报告、参数扫描报告和因子贡献报告，便于后续 API 对接和前端展示。

## 三、改进建议

### 3.1 回测区间选择需谨慎考量

**问题**：方案选择 2024-2025 年作为回测区间，但 2024 年市场经历了显著的风格切换和结构性行情。若仅基于这两年数据进行优化，可能存在以下风险：

- **样本偏差**：2024-2025 年 A 股市场具有特殊性（如政策密集出台、资金结构变化等），策略可能过度适应短期市场特征
- **周期覆盖不足**：两年数据难以覆盖完整的经济周期和牛市/熊市转换

**建议**：

- 考虑将回测区间扩展至 2019-2025 年（覆盖疫情前后、复苏期、结构市等不同阶段）
- 若历史数据有限，至少应在报告中明确说明回测区间的局限性
- 可增加「分年度表现」分析，观察策略在不同年份的稳定性

### 3.2 缺少基准对比维度

**问题**：方案设定基准为「沪深 300」，但在评估指标中未明确说明与基准的超额收益（Alpha）情况。对于稳健型策略而言，**稳定的超额收益**比绝对收益更重要。

**建议**：

- 在评估指标中增加「年化超额收益」「信息比率」等基准对比指标
- 对于稳健型策略，可考虑增加与纯多头指数基金的对比分析

### 3.3 参数扫描空间可进一步细化

**问题**：当前融合权重仅设置 3 档（保守/均衡/积极），持仓天数 4 档，止损阈值 4 档，网格组合数为 3×4×4×4=192 种。对于稳健型策略，建议增加更多中间档位以提升精细度。

**建议**：

- 融合权重可增加更多档位，如：cp=0.45/0.4/0.35/0.3，gain=0.35/0.3/0.4/0.45，prob=0.2/0.3/0.25/0.25
- 持仓天数可考虑 4/5/6/7，止损阈值可考虑 -8%/-10%/-12%/-15%
- 可考虑增加「持仓集中度」作为扫描维度（如 Top5/Top6/Top8）

### 3.4 因子分析维度可扩展

**问题**：当前因子贡献分析仅采用「逐步剔除法」，方法相对单一。逐步剔除法虽然直观，但存在以下局限：

- 无法捕捉因子间的**交互效应**
- 未考虑因子在不同市场环境下的**时变特性**

**建议**：

- 增加 **IC 分析**（信息系数），评估因子的预测能力稳定性
- 增加 **分组回测**分析，验证因子的单调性（高因子值是否持续优于低因子值）
- 可引入 **SHAP 值**或 **特征重要性**分析，更精准地量化因子贡献

### 3.5 缺少交易成本和滑点模拟

**问题**：方案回测参数中未明确提及交易成本（佣金、印花税等）和滑点设置。对于集中持仓策略，交易成本的影响尤为显著。

**建议**：

- 在回测配置中明确交易成本参数（如：佣金万 2.5、印花税千 1、滑点 0.1%）
- 对于稳健型策略，可增加敏感性分析，观察交易成本对最终收益的影响

### 3.6 缺少风控模块设计

**问题**：方案聚焦于策略收益优化，但对风控模块的设计着墨较少。稳健型策略的核心之一就是风险控制，当前方案仅通过止损和最大回撤容忍来约束，略显单薄。

**建议**：

- 增加 **动态仓位调整**机制（如根据市场波动率调整仓位）
- 增加 **行业/市值中性**约束，避免集中度过高
- 增加 **最大单日亏损**限制（如单日亏损超过 3% 自动减仓）

### 3.7 策略类型可补充

**问题**：当前测试的 6 种策略以单因子和简单多因子为主，对于「稳健型」定位，可考虑增加更具防御性的策略类型。

**建议**：

- 增加 **低波动策略**（按波动率排序，选取波动最低的股票）
- 增加 **高股息策略**（按股息率排序）
- 增加 **价值成长平衡策略**（growth×0.25 + value×0.25 + quality×0.3 + momentum×0.2）

## 四、风险提示

### 4.1 过拟合风险

参数扫描阶段生成的 192 种组合中筛选最优，可能导致策略过度适应历史数据。需通过测试集验证和样本外跟踪来控制。

### 4.2 市场风格切换风险

2024-2025 年市场以结构性行情为主，趋势类策略可能表现较好。若未来市场转为震荡或快速轮动，当前策略可能面临回撤加大。

### 4.3 流动性风险

集中持仓 5-8 只，单只仓位 8-15%，若持仓股票出现流动性问题（如连续跌停），可能难以有效止损。
