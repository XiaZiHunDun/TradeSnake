## 二、关于现金战力定义的优化建议

### 报告中的方案

报告建议：现金战力 = 无风险收益/年化2%

### 本小姐的补充

本小姐建议采用**更精细的动态模型**：

```python
class CashCP:
    """现金战力计算"""
    risk_free_rate = 0.02  # 年化无风险利率 2%

    @staticmethod
    def get_opportunity_cost(cash: float, days: int) -> float:
        """
        持有现金的每日机会成本战力

        公式：cash × (2% / 365) × days
        例：10万现金持有30天 = 100000 × 0.000055 × 30 = 165 战力
        """
        daily_rate = CashCP.risk_free_rate / 365
        return cash * daily_rate * days
```

**战力含义**：持有现金的"损失"就是看着优质股票涨，自己没买

### 三种方案的对比

| 方案 | 现金战力 | 优点 | 缺点 |
|------|---------|------|------|
| A | = 0（最低基准） | 简单 | 忽略机会成本 |

## 三、关于战力差阈值的精细化

### 报告中的公式

```
换股净收益 = (B战力 - A战力) × 本金 - 交易成本
```

### 本小姐建议增加"持有时间"维度

```
换股净收益 = (B战力 - A战力) × 本金 × 持有天数 - 交易成本
```

**理由**：
- 如果今天买入明天就卖，持有天数=1天，战力差需要很大才划算
- 如果打算持有30天，战力差可以小一点

### 战力差阈值计算（修正版）

| 资金 | 交易成本 | 持有30天阈值 | 持有60天阈值 | 持有90天阈值 |
|------|----------|-------------|-------------|-------------|
| 1万 | ~10元 | 3.3%/月 | 1.7%/月 | 1.1%/月 |
| 5万 | ~56元 | 0.37%/月 | 0.19%/月 | 0.12%/月 |
| 10万 | ~112元 | 0.37%/月 | 0.19%/月 | 0.12%/月 |

**结论**：
- 小资金做短线几乎不可能赚钱（被交易成本吃掉）
- 资金越大、持有越久，阈值越低
- **这个分析要明确告诉用户！**
