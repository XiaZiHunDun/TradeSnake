# 股市贪吃蛇 (TradeSnake) - 初始设计文档

## 一、项目愿景

将股票市场重新诠释为一场**贪吃蛇游戏**。

传统的炒股思维是"低买高卖赚差价"，而本项目的核心理念是：**股市不是一个"赚钱"的地方，而是一个"价值置换与进化"的战场**。

> 类比：你的初始资金不是"钱"，而是一股**初始战力**。股市中每一只股票都有其"战力值"。你的目标不是持有更多金钱，而是通过不断**置换**——将低战力股票置换为高战力股票——来**持续提升自己的总战力**，最终成为战力最强的贪吃蛇。

---

## 二、核心概念映射

| 贪吃蛇游戏 | 股市实践 | 说明 |
|-----------|---------|------|
| 贪吃蛇 | 投资者自身 | 进入市场的交易者 |
| 蛇身长度 | 总战力值 | 衡量投资者实力的核心指标 |
| 食物 | 高价值股票 | 战力提升的来源 |
| 蛇头方向 | 市场趋势 | 指引贪吃蛇前进的方向 |
| 撞墙/撞自己 | 亏损/被套 | 游戏失败的代价 |
| 地图 | 股票市场全貌 | 所有可交易股票的集合 |
| 蛇生长 | 战力提升 | 通过成功置换实现 |
| 蛇缩短 | 战力下降 | 持有贬值股票 |

## 三、核心价值体系

### 3.1 战力值 (Combat Power / CP)

**战力值**是本项目的核心价值计量单位。

- 每只股票都有其**股票战力值 (Stock CP)**
- 投资者的**个人战力值 = 持有股票的战力值之和**
- 战力值不再以金钱衡量，而是基于股票的**内在价值因子**计算

### 3.2 战力值计算逻辑（初步设想）

```
股票战力值 = 基础因子 × 成长因子 × 稀缺因子

- 基础因子：股票的基本面价值（PE、ROE、营收增长率等）
- 成长因子：股票的成长性指标（净利润增速、市场份额变化）
- 稀缺因子：股票的稀缺程度（行业壁垒、市场地位、不可替代性）
```

### 3.3 置换机制

**置换**是本项目的核心操作：

```
低战力股票 + 手续费/时间成本 → 高战力股票
```

- **同板块置换**：同一行业内，将战力低的股票换成战力高的
- **跨板块置换**：将整个板块视作一个战力单位，进行板块间置换
- **逆势置换**：在市场下跌时，将账面战力暂时降低，换入未来潜力更高的股票

## 四、贪吃蛇的价值判定规则

### 4.1 最具战力股票榜单

- 系统实时计算并展示**战力榜 TOP 100**
- 战力榜每分钟/每日更新（可配置）
- 战力榜揭示了市场中最"值得置换"的目标

### 4.2 战力趋势

- 每只股票有**战力趋势图**：上升↗、平稳→、下降↘
- 战力上升的股票是"食物"
- 战力下降的股票是"毒药"

### 4.3 战力对比

- 个股详情页展示该股票与同行业其他股票的战力对比
- 类似于贪吃蛇游戏中"前方有更大的蛇/食物"的提示

## 五、用户体验设计

### 5.1 地图视图

将股市呈现为一个**贪吃蛇游戏地图**：

- **整个A股/港股/美股市场**是游戏的棋盘
- 每只股票是一个**格子**，战力越高的股票"颜色越亮/越显眼"
- 投资者是一个**贪吃蛇图标**，在地图上移动
- 移动路径展示历史操作

### 5.2 操作方式

| 游戏操作 | 对应股市操作 |
|---------|-------------|
| 吞下食物 | 买入股票 |
| 蛇身变长 | 战力提升 |
| 撞墙死亡 | 止损/被套牢 |
| 蛇身变短 | 战力下降/割肉 |
| 加速 | 重仓/全仓操作 |
| 减速 | 轻仓观望 |

### 5.3 视觉反馈

- **战力提升**：特效动画 + 蛇身变长动画
- **战力下降**：红色闪烁 + 蛇身缩短动画
- **成功置换**：两个格子交换位置的动画
- **战力排行榜**：实时滚动的榜单，类似贪吃蛇长度排行榜

## 九、战力公式详细设计

### 9.1 v1.0 简化公式（先跑通）

```
战力值 = 成长分 × 40 + 价值分 × 30 + 趋势分 × 30
```

各分数标准化到 0-100 分范围。

| 因子 | 权重 | 数据来源 | 计算方式 |
|-----|------|---------|---------|
| **成长分** | 40% | 财报 | (净利润增速 + 营收增速) / 2，标准化 |
| **价值分** | 30% | 实时行情 | (1/PE + ROE) / 2，标准化 |
| **趋势分** | 30% | 行情数据 | 近20日涨跌幅 + 成交量变化，标准化 |

**注意**：
- PE为负或过高的股票，价值分取低值
- 上市不足1年的新股，成长分用营收增速替代
- 标准化方式：百分位排名（该股票在A股的排名百分比 × 100）

---

### 9.2 公式参数可调区

```
# 后续可通过配置文件调整权重
WEIGHTS = {
    "growth": 0.4,
    "value": 0.3,
    "momentum": 0.3
}

# 或者预设几种模式
MODES = {
    "激进": {"growth": 0.6, "value": 0.2, "momentum": 0.2},
    "平衡": {"growth": 0.4, "value": 0.3, "momentum": 0.3},
    "保守": {"growth": 0.2, "value": 0.5, "momentum": 0.3}
}
```

---

### 9.3 个人战力计算

```
个人总战力 = Σ(持有股票数量 × 该股票战力值)
```

也就是说：你持有100股某股票，该股票战力值80，那这只股票对你的战力贡献就是 100 × 80 = 8000。

**战力变化** = 今日战力 - 昨日战力（类似于贪吃蛇的长度变化）

### 10.4 数据输入方式（v1.0）

因为是私人工具，v1.0采用**手动录入持仓**的方式：

```
持仓管理页面：
┌─────────────────────────────────────┐
│ 我的持仓                             │
│ ┌────┬─────┬──────┬─────┐           │
│ │名称│代码 │持股数│成本价│           │
│ ├────┼─────┼──────┼─────┤           │
│ │宁德│300750│ 100  │180.5│ [编辑]   │
│ │茅台│600519│  50  │1600 │ [编辑]   │
│ └────┴─────┴──────┴─────┘           │
│                                     │
│ [+ 添加持仓]                         │
└─────────────────────────────────────┘
```

**为什么不自动获取持仓？**
- 需要对接券商API，私人工具不值得
- 手动录入更可控，也符合"思考投资"的目的

| 层级 | 推荐技术 | 理由 |
|-----|---------|------|
| **数据源** | 东方财富/同花顺 | A股数据全，本地即可获取 |
